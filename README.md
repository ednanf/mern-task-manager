## REST API routes

| METHOD | ROUTE             | DESCRIPTION     |
|--------|-------------------|-----------------|
| GET    | /api/v1/tasks     | get all tasks   |
| POST   | /api/v1/tasks     | create task     |
| GET    | /api/v1/tasks/:id | get single task |
| PATCH  | /api/v1/tasks/:id | update task     |
| DELETE | /api/v1/tasks/:id | delete task     |
